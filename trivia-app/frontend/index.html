Welcome to the GitOps Trivia App!

1. What is GitOps?

A way to manage infrastructure using Git as the source of truth.

2. What are GitOps principles?

Declarative, version-controlled, automated, and monitored.

3. Why use GitOps?

For consistency, automation, and quick rollbacks.

4. What tools support GitOps?

ArgoCD, Flux, Jenkins X, etc.

5. How does GitOps manage rollbacks?

By reverting to previous Git commits.

6. Difference between CI/CD and GitOps?

CI/CD triggers pipelines, GitOps syncs directly from Git.

7. What is ArgoCD?

A tool to deploy apps via Git in Kubernetes.

8. How does GitOps ensure consistency?

By reconciling actual vs. desired states.

9. Is GitOps secure?

Yes, changes are tracked in Git with clear audit trails.

10. Can GitOps work outside Kubernetes?

Yes, with appropriate tools for other environments.
